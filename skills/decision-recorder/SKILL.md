# Decision Recorder

Use when a meaningful implementation decision is made.

## Template
- Title: `Decision: <topic>`
- Date
- Options considered
- Decision
- Trade-off
- Follow-up

## File Location
- `decision-docs/DEC-XXX-<slug>.md`
- Increment `DEC-XXX` sequentially.
